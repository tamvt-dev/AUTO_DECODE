# HyperDecode Branding

Files in this folder:

- `hyperdecode-mark.svg`: square app mark for icon work and installer branding
- `hyperdecode-app-icon.svg`: simplified icon optimized for small Windows sizes
- `hyperdecode-logo.svg`: horizontal logo for README, splash art, and release assets

Design direction:

- minimal geometric monogram
- dark technical shell with a warm neutral wordmark
- neon green decode brackets and a blue signal core

Color palette:

- `#0A0E27`
- `#121938`
- `#252D4A`
- `#37F712`
- `#00A6F4`
- `#F4E7DA`
