Auto Decoder Pro - Portable Version
====================================

SYSTEM REQUIREMENTS
- Windows 7/8/10/11 (64-bit)
- 50MB free disk space
- No installation required!

HOW TO USE
1. Extract the ZIP file to any folder
2. Double-click "run_portable.bat"
3. Application will start with all settings saved in the same folder

FOLDER STRUCTURE
AutoDecoderPro/
├── auto_decoder_pro.exe    - Main GUI application
├── auto_decoder.exe        - Command-line version
├── run_portable.bat        - Launcher script
├── resources/              - Icons and themes
└── data/                   - User data (created on first run)

COMMAND LINE
Open CMD in this folder and run:
  auto_decoder "SGVsbG8gV29ybGQ="
